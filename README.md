## PROJECT OVERVIEW
- The system is designed to recommend welfare schemes based on a user's personal details such as income, category, gender, age, educational level, rural background, and disability status.
- The idea behind this project is simple:
  Many people are not aware of the welfare schemes they are eligible for. The information is often scattered and difficult to understand. This system demonstrates how logical reasoning and knowledge representation can be used to build a simple decision-support system.
- Instead of manually checking eligibility conditions, the system applies predefined logical rules and automatically suggests suitable schemes.

## HOW THE SYSTEM WORKS:
The logical flow of the system is:
User Input → Knowledge Storage → Rule Matching → Backtracking → Explanation Generation → Priority Classification

1. The system asks the user to enter personal details.
2. User information is stored as structured facts.
3. Income is classified into categories (low, middle, high).
4. Logical rules check eligibility conditions.
5. All matching welfare schemes are displayed.
6. The system provides reasons for eligibility.
7. A priority level (P1, P2, or P3) is assigned based on inclusion criteria.
